## üöÄ Preparing the Alpha Package

### Step 1: Clean the Directory

Remove generated files and caches:
```bash
# Remove database files
rm -f *.db
rm -rf instance/
rm -rf chroma_db/

# Remove logs
rm -f *.log*
rm -rf logs/

# Remove caches
find . -type d -name "__pycache__" -exec rm -rf {} +
find . -type d -name "node_modules" -exec rm -rf {} +
find . -type f -name "*.pyc" -delete

# Remove virtual environment
rm -rf .venv/

# Remove user-specific files
rm -f user_settings.json

# Remove uploads
rm -rf uploads/
```

### Step 2: Verify Documentation

Ensure these docs are up-to-date:
- [ ] README.md - Updated for alpha users
- [ ] QUICK_START.md - Clear first-time instructions
- [ ] docs/DATABASE_SETUP_GUIDE.md - Database setup
- [ ] docs/gettin_started/ - Getting started guides

### Step 3: Check Credentials

Ensure NO credentials are included:
- [ ] `app/assistant/lib/credentials/` should be empty or have only README
- [ ] No `token.pickle` or `credentials.json` files
- [ ] No `.env` files with secrets
- [ ] No `app/certs/*.pem` files (or only self-signed for localhost)

### Step 4: Create Package

Create a zip/tar file with only the necessary files:

```bash
# Using git to create clean package (respects .gitignore)
git archive -o EmiAi_alpha_v1.0.zip HEAD

# Or manually create archive excluding unwanted directories
tar -czf EmiAi_alpha_v1.0.tar.gz \
  --exclude='_archive' \
  --exclude='postgresql_to_sqlite_migration' \
  --exclude='migration_scripts' \
  --exclude='tests' \
  --exclude='sqlite_utilities' \
  --exclude='tools' \
  --exclude='*.db' \
  --exclude='chroma_db' \
  --exclude='instance' \
  --exclude='logs' \
  --exclude='uploads' \
  --exclude='.venv' \
  --exclude='__pycache__' \
  --exclude='node_modules' \
  --exclude='*.pyc' \
  --exclude='*.log' \
  --exclude='user_settings.json' \
  .
```

## üö¢ Deployment Instructions for Testers

Include a `INSTALL.md` file in the package:

```markdown
# EmiAi Alpha Installation

## Prerequisites
- Python 3.10 or higher
- 4GB RAM minimum
- 2GB disk space

## Installation Steps

1. Extract the archive
2. Open terminal in the extracted directory
3. Run: `python setup.py`
4. Follow the prompts
5. Activate virtual environment:
   - Windows: `.venv\Scripts\activate`
   - Mac/Linux: `source .venv/bin/activate`
6. Run: `python run_flask.py`
7. Open browser to: http://localhost:5000
8. Complete setup wizard

See QUICK_START.md for detailed instructions.
```
